■1.目的
本ツールは、青々とした緑色の木々の写真に対して、
RGBの緑色要素を赤色要素に置き換えることで、
手軽に紅葉の写真のように変換するソフトウェアです。


■2.実行環境
Windows10 64bit + .net Framework4.6.2以降が必要です。


■3.使い方
AutumnEditor.exeを起動します。

1.の欄で、編集したい写真ファイルを選択します。
3.EDITのボタンを押します。
